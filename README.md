# mekkz AI

Premium AI Assistant Web App built with Next.js, Supabase, and OpenAI.

## Tech Stack
- Next.js 15 + React 19 + TypeScript
- Tailwind CSS + Framer Motion
- Supabase (Auth + Postgres + Storage)
- OpenAI API (Claude-ready adapter)

## Quick Start
1. Install dependencies:
   - `npm install`
2. Copy env file:
   - `cp .env.example .env.local`
3. Fill environment variables.
4. Run database SQL from `supabase/schema.sql`.
5. Start development:
   - `npm run dev`

## Deploy
- Vercel (recommended)
- Add all env vars in Vercel project settings.
